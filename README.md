# 🏭 Monitoramento Preditivo de Motores Elétricos com Machine Learning

Projeto de Ciência de Dados e IoT focado na Indústria 4.0. Este sistema utiliza um modelo de Machine Learning (Random Forest) para analisar dados de vibração e temperatura e prever a probabilidade de falha de um motor elétrico nas próximas 24 horas.
